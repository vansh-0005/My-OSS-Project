Project Overview
This repository contains my Capstone Project for the OSS course. It focuses on a structured audit of Git, exploring its origins, licensing, and its role in the modern FOSS ecosystem.

The project consists of two main parts:

The Audit Report: A detailed PDF documenting the software's history, footprint, and ethical implications.

Shell Automation: Five Bash scripts that demonstrate practical Linux interaction and system auditing.

Repository Structure
/OSSCapstoneProject.pdf: The full 12-16 page audit report.

script1.sh: Generates a system identity report (Script 1).

script2.sh: Checks for Git installation and prints its philosophy (Script 2).

script3.sh: Handles automated file backups (Script 3).

script4.sh: Parses system logs for specific patterns (Script 4).

script5: A creative script reflecting on open-source freedom (Script 5).
